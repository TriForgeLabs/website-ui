Data Retention and Account Deletion

TriForgeLabs applications only retain the minimum data necessary for user accounts. This data may include, but is not limited to, user's names, emails, account identifiers, and any provided anthropometric information.

All of the user's data is removed if the user account is deleted. If you wish to delete your account, please follow these steps:

  1. Open the application on your mobile device.
  2. Navigate to the Settings page.
  3. Select Delete Account.
  4. Confirm in the popup that you would like to delete the account.

Important: When an account is deleted, all data pertaining to that account is purged. Deleting an account is permanent and cannot be undone or recovered after deletion.
